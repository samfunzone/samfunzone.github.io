# Kids Fun Zone

A colorful, interactive React app packed with games and activities for kids.

## Features

| Tab | Description |
|-----|-------------|
| 🦔 Whack-a-Mole | Hit the moles before the 30-second timer runs out |
| 🎈 Balloon Pop | Click rising balloons before they fly away |
| 🧠 Memory Match | Flip cards to find all 8 emoji pairs |
| 😂 Joke Machine | Two-step kid-friendly joke reveal |
| 🎨 Drawing | Freehand canvas with colors, brush size, undo, and save |

## Getting Started

```bash
npm install
npm run dev
```

App runs at **http://localhost:3000**.

## Stack

- React 19 + Vite
- Plain CSS (no UI library)
